Eye Of The Beholder 3 - Fan Patched Retro Dungeon Crawler


Another great dungeon crawler rpg in the trilogy of Eye of the Beholder, was number 3, the Assault on Myth Drannor. Unlike the previous games it was not developed by Westwood Studios but in-house by the publisher SSI in 1993/4. The third and final game of the Eye of the Beholder series would see players embarking on a quest to the fabled ruins of the city of Myth Drannor. In my opinion it was still a worthy dungeon crawler, but many fans of the series did not like it at all, as it had over sized maze like maps, performance problems, unchallenging battles with poorly designed puzzles and the storyline wasn't so good. But yet I still enjoyed it, and today you can too, with a lot of those performance problems fan fixed!


Eye of the Beholder 3 fan patched version is running with the AESOP/32 instead of the AESOP/16 engine. It makes the difference between it running unbearably choppy and now almost perfectly smooth. Original thread and fanbased coder info here

DOSBOX WIKI GUIDE

2) Load the DOSBox 0.74 Options.bat or edit the dosbox.conf file with the settings below, don't copy and overwrite the entire list. Just edit the relevant options, for example memsize=64.

[sdl]
fullscreen=true ( False if you want to be windowed )
fulldouble=false
fullresolution=desktop
windowresolution=1024x768
output=ddraw
autolock=true
sensitivity=100
waitonerror=true
priority=higher,normal
mapperfile=mapper-0.74.map
usescancodes=true

[dosbox]
language=
machine=svga_s3
captures=capture
memsize=64

[render]
frameskip=0
aspect=false
scaler=super2xsai

[cpu]
core=dynamic
cputype=auto
cycles=max
cycleup=10
cycledown=20

[sblaster]
sbtype=sb16
sbbase=220
irq=7
dma=1
hdma=5
sbmixer=true
oplmode=auto
oplemu=default
oplrate=44100

[dos]
xms=true
ems=true
umb=true
keyboardlayout=auto

[ipx]
ipx=false

[autoexec]
mount c C:\DOS\  (Make a directory called C:\DOS\ using windows )
C:
imgmount d ".\games\eob3\cd\Eye of the Beholder Trilogy.iso" -t cdrom (IGNORE IF USING UNPACKED ZIP FILES)

* SAVE THE FILE *

2) Download Eye of the Beholder 3(search for it, it's abandoned) or if you already have it then install using dosbox, ie make sure all main files are in C:\DOS\EOB3

3) Download the fanbased files, which i've put together to make it easier for gamers! Then unpack overwrite using winrar/zip into your EOB3 main folder.

4) Load up dosbox again, go into your EOB3 folder  and run sound and make sure it's set to standard soundblaster.

5) Save that, run EYE and have fun!
