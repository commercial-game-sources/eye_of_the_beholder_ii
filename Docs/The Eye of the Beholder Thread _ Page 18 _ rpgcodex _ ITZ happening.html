    I also saw one GOG forum post which made me think that the GOG-version is unpatched. But maybe they have updated the package?

    Downloaded the runtime.zip from here:
    http://www.vogons.org/viewtopic.php?t=20601

    AESOP.EXE
    DOS4GW.EXE
    INTERP.EXE
    A32ADLIB.DLL
    A32MT32.DLL
    A32SBDG.DLL
    A32SPKR.DLL
    MCGA.DLL

    The GOG-versions has these exact files included. Abandonware versions mostly don't.

    One abandonware version has an EYE.RES file which is just 4.2 megs. I patched the EYE.RES file and it outputted a 6.5 megabyte file which isn't exactly the same size as the GOG-version but close. The EYE.RES file contains datestamps so they might not be comparable even if they're the same size. The patched EYE.RES still has AESOP/16 identifier in the beginning of the file.

    The unpatched abandonware version was noticeably slower in menus and especially slow when loading saved games. The GOG-version seems to run even too fast on default DOSBox-settings but it's also surprisingly demanding for a DOSBox game even on a modern computer.

         7-14                 0
